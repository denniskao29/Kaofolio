รายละเอียดเกี่ยวกับเว็บไซต์

เว็บไซต์นี้เป็นเว็บไซต์แฟ้มสะสมผลงานของผมที่ถูกพัฒนาโดยใช้ Visual Studio Code ซึ่งเป็นเครื่องมือพัฒนาเว็บไซต์ที่มีประสิทธิภาพสูง โดยผมได้ใช้ ภาษา HTML และ CSS ในการสร้างโครงสร้างและดีไซน์ของเว็บไซต์ทั้งหมด นอกจากนี้ การใช้ Generative AI ในการพัฒนาเว็บไซต์นี้ยังช่วยเสริมประสิทธิภาพในการออกแบบและการสร้างเนื้อหาต่าง ๆ ได้อย่างรวดเร็วและมีความแม่นยำมากยิ่งขึ้น โดยการใช้ AI สามารถสร้างเนื้อหาหรือกราฟิกบางประเภทที่จำเป็นสำหรับเว็บไซต์ได้ เช่น การสร้างข้อความอัตโนมัติสำหรับการอธิบายผลงานหรือการสร้างภาพประกอบที่มีลักษณะเฉพาะตามที่ต้องการ โดยสามารถปรับแต่งให้เหมาะสมกับคอนเซ็ปต์ของเว็บไซต์ ในช่วง 3 ปีที่ผ่านมา เพื่อแสดงผลงานต่าง ๆ ที่ผมได้สร้างขึ้นในระยะเวลานั้น โดยมีทั้งงานออกแบบกราฟิก การพัฒนาเว็บไซต์ และการใช้งานเทคโนโลยีดิจิทัลต่าง ๆ โดยทำเป็นแฟ้มสะสมผลงานแบบสำรองที่เก็บผลงานรายละเอียดเยอะ เพื่อยื่นเข้าสาขาสื่อศิลปะและการออกแบบ คณะวิจิตรศิลป์ มหาวิทยาลัยเชียงใหม่
